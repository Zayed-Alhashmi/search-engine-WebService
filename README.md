# Search Engine — COMP3011 Coursework 2

A command-line search engine tool that crawls, indexes, and searches a website.

## Project Structure
- `src/crawler.py` — Web crawler
- `src/indexer.py` — Inverted index builder
- `src/search.py` — Search and query processing
- `src/main.py` — Command-line interface
- `tests/` — Test suite
- `data/` — Compiled index file (generated by the build command)

## Setup
```bash
pip install -r requirements.txt
```
